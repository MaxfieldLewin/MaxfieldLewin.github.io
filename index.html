Projects

Savant

Savant is a collaborative text-annotation platform, inspired by Genius. Any user can create public annotations on texts uploaded by any other user by simply highlighting the given portion of text in the browser. Savant is built on a RESTful JSON Api implemented in Rails, with the frontend implemented as a single-page Backbone.js app.

Rails Lite

Rails Lite is a Ruby web server implmementing a limited subset of Ruby on Rails' ActiveController library, including request parameters parsing, dynamically generated routes and routing helper methods for both backend logic and html generation, and template parsing and instantiation.
